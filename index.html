Processo de Comunicação

A comunicação de dados é a troca de informações entre dispositivos através de um meio de transmissão. Requer cinco componentes: Emissor, Receptor, Mensagem, Meio de Transmissão e Protocolo.

A Corrente do Bem

O desenvolvimento das redes foi impulsionado por 3 frentes: IBM (Mainframes, time-sharing), DoD dos EUA (ARPANET, TCP/IP) e Xerox PARC (Ethernet).

Conceitos Formais

Redes de Computadores

Conjunto de computadores autônomos interconectados para compartilhamento de recursos.

Sistemas Distribuídos

Coleção de computadores independentes que se apresenta ao usuário como um único sistema coerente, visando desempenho e confiabilidade.

Topologias de Rede

A Topologia Física é o layout dos cabos, enquanto a Topologia Lógica é como os dados fluem. Principais físicas: Anel (convencional, Token Ring, FDDI com anel duplo), Barramento, Estrela, Malha e Árvore.
